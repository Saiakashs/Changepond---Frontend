1. <!DOCTYPE html>- It tells search engine this is html type of document belogns to version 5.
2. head - It contains external links and meta data in it.

5. <meta charset="UTF-8">- it converts the code into machine readable input
6. <meta name="viewport" content="width=device-width, initial-scale=1.0">- This code helps to adapt webpage to different size devices

In HTML  there are two types of element
1.Block level element: It doesn't allow other tag contents on its tag space or line. Example: br,h1-h6,p,li,table,form
2.Inline level element: It allows other tag contents on its tag space or line. Example: img,a,b,i,u

Form:

The form is used to submit the user data. We use two methods for that.

GET:

1.The user data will be visible in browser url.
2.You are not prefered to use this method for submitting confidential data.
3.There is character limitation in sending the data through get method.

POST:

1.The user data will not  be visible in browser url.
2.You are prefered to use this method for submitting confidential data.
3.There is no character limitation in sending the data through POST method.
